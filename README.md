# Inventory Management System

Full-stack inventory management system built with Python and JavaScript.

## Features
- User authentication & login system
- Advanced product search (name, price range)
- Category management
- Real-time inventory tracking
- Stock calculations & reporting
- Data persistence with file storage
- Responsive design

## Tech Stack
- **Backend:** Python
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** File-based (TXT)

## Installation
1. Clone the repo
2. `python inventario.py`
3. Login: admin / 123

## Usage
- Add categories
- Add/modify products
- View inventory with filters
- Calculate total value
- Generate reports
